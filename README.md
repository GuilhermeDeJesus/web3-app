# Web3 Clean Architecture App

This project is an example of MetaMask integration using React + Vite, following Clean Architecture, SOLID, Aggregate, and DDD principles.

## How to run

1. Install dependencies:
   Run in terminal:
   npm install
   
2. Start the project:
   Run in terminal:
   npm run dev
   

## Structure

- `src/domain`: Entities and business rules
- `src/application`: Use cases
- `src/infrastructure`: External services (e.g., Web3)
- `src/presentation`: React UI
- `docs/`: Documentation and diagrams

## Features
- Detects MetaMask
- Allows wallet connection
- Shows connected address
- Updates on account change
- Allows disconnecting the wallet

See more in [docs/ARQUITETURA.md](docs/ARQUITETURA.md)
